Place mascot PNG files in this folder with exactly these names:

- ayushman-mascot-primary.png
- ayushman-mascot-happy.png
- ayushman-mascot-thinking.png
- ayushman-mascot-alert.png

These are used by `src/components/MascotAvatar.tsx`.
If files are missing, the app automatically falls back to the built-in SVG mascot.
